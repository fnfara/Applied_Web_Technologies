▶ HTML История (нажмите)

HTML был создан Тимом Бернерсом-Ли в 1991 году. Первая версия содержала всего 18 тегов. HTML5 был опубликован как рекомендация W3C в 2014 году.


▶ Версии HTML
  1. HTML 1.0 — 1991
  2. HTML 2.0 — 1995
  3. HTML 3.2 — 1997
  4. HTML 4.01 — 1999
  5. XHTML 1.0 — 2000
  6. HTML5 — 2014
  7. HTML Living Standard — ныне
