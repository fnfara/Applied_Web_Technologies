1. Введение в HTML

Что такое HTML?

HTML (HyperText Markup Language) — язык разметки гипертекста, который является основой всех веб-страниц в интернете. HTML был создан Тимом Бернерсом-Ли в 1991 году.

HTML — это не язык программирования, а язык разметки. Он описывает структуру и содержание веб-страницы с помощью специальных меток — тегов.

Синтаксис тегов

Тип тега Синтаксис Пример
Парный (открывающий + закрывающий) <тег>содержимое</тег> <p>Текст</p>
Одиночный (самозакрывающийся) <тег> или <тег /> <br> <img> <hr>
Тег с атрибутом <тег атрибут="значение"> <a href="url">

💡 Важно: HTML не чувствителен к регистру — <P> и <p> одно и то же. Однако рекомендуется писать теги строчными буквами.

2. Теги для работы с документом
<!DOCTYPE>  — объявление типа документа

Первый тег в любом HTML-документе. Сообщает браузеру, какую версию HTML использовать. Не является обычным тегом — это инструкция для браузера.

📄 Пример:
<!DOCTYPE html>    <!-- HTML5 -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN">    <!-- HTML 4.01 -->
<html> ... </html>  — корневой элемент

Главный контейнер страницы. Всё содержимое HTML-документа размещается внутри этого тега.

Атрибут Описание Пример
lang Язык документа lang="ru"
xmlns Пространство имён (для XHTML) xmlns="http://www.w3.org/1999/xhtml"
manifest Файл манифеста для офлайн-приложений manifest="app.appcache"
<head> ... </head>  — невидимая часть документа

Содержит служебную информацию о документе. Содержимое тега <head> не отображается на странице. Внутри размещаются: <title>, <meta>, <link>, <style>, <script>.

📄 Пример:
<head>
  <meta charset="UTF-8">
  <title>Название страницы</title>
  <link rel="stylesheet" href="style.css">
</head>
<body> ... </body>  — видимая часть страницы

Всё, что видит пользователь на странице, находится внутри тега <body>. Это основной контейнер для отображаемого содержимого.

Атрибут Описание
bgcolor Цвет фона страницы
background Фоновый рисунок
text Цвет текста
link Цвет непосещённых ссылок
vlink Цвет посещённых ссылок
alink Цвет активной (нажатой) ссылки
leftmargin / rightmargin / topmargin / bottommargin Отступы от краёв окна

Семантические теги HTML5

В HTML5 появились теги для более осмысленной разметки страницы. Они помогают браузерам, поисковым системам и программам чтения с экрана лучше понимать структуру страницы.

Тег Назначение Пример использования
<header> Шапка страницы или раздела (логотип, заголовок, меню) <header><h1>Мой сайт</h1></header>
<footer> Подвал страницы (автор, контакты, копирайт) <footer><p>© 2024</p></footer>
<nav> Навигационное меню с основными ссылками <nav><a href="#">Главная</a></nav>
<section> Тематический раздел страницы с заголовком <section><h2>О нас</h2>...</section>
<article> Самостоятельная статья, пост или новость <article><h2>Новость</h2>...</article>
<aside> Боковая панель (сайдбар) с дополнительным контентом <aside><p>Реклама</p></aside>

📄 Типичная структура страницы HTML5:

<body>
 <header> ... </header>
 <nav> ... </nav>
 <section>
  <article> ... </article>
  <aside> ... </aside>
 </section>
 <footer> ... </footer>
</body>

5. Теги для работы со ссылками

Тег <a> — гиперссылка

Тег <a> (anchor — якорь) является одним из важнейших в HTML. Создаёт гиперссылку — текст или изображение, кликнув на которое можно перейти на другую страницу или ресурс.

Тип ссылки Код Результат
Внешняя ссылка <a href="https://google.com">Google</a> Google
Открыть в новой вкладке <a href="https://..." target="_blank">Ссылка</a> Ссылка (новая вкладка)
Ссылка на email <a href="mailto:[email protected]">Написать</a> Написать нам
Ссылка на телефон <a href="tel:+79001234567">Позвонить</a> +7 (900) 123-45-67
Якорная ссылка (к разделу) <a href="#section-doc">К разделу</a> К разделу 2 ↑
Изображение-ссылка <a href="..."><img src="pic.jpg"></a> (изображение кликабельно)

Атрибут target

Значение Поведение
_blank Открыть в новой вкладке/окне
_self Открыть в текущей вкладке (по умолчанию)
_parent Открыть в родительском фрейме
_top Открыть в полном окне, отменяя все фреймы
имя_фрейма Открыть в указанном именованном фрейме

6. Теги для работы с медиа

Тег <img> — Изображения

Тег <img> используется для вставки изображений. Поддерживает форматы: JPEG, PNG, GIF, WebP, SVG, BMP.

Атрибут Описание Обязательный?
src Путь к файлу изображения ДА
alt Альтернативный текст (если изображение не загрузилось) ДА
width Ширина изображения (в пикселях или %) нет
height Высота изображения нет
title Всплывающая подсказка при наведении нет
border Рамка вокруг изображения нет
hspace / vspace Горизонтальный / вертикальный отступ нет

📄 Пример кода:

<img src="photo.jpg" alt="Описание фото" width="300" height="200">

<!-- Изображение-ссылка -->
<a href="page.html">
  <img src="button.png" alt="Кнопка" border="0">
</a>

Тег <audio> — Аудио (HTML5)

Атрибут Описание
src Путь к аудиофайлу (.mp3, .ogg, .wav)
controls Показать панель управления (кнопки воспроизведения)
autoplay Автоматическое воспроизведение при загрузке
loop Зациклить воспроизведение
muted Начать без звука
preload Предзагрузка: auto | metadata | none

<audio controls>

  Ваш браузер не поддерживает аудио.
</audio>

Атрибут Описание

width / height Размеры видеоплеера
poster Изображение-превью до начала воспроизведения
autoplay Автовоспроизведение
loop Повтор
muted Без звука (нужно для autoplay в современных браузерах)

7. Теги для работы с таблицами

Таблицы в HTML создаются с помощью нескольких тегов, работающих вместе. Основной тег — <table>, внутри которого строки (<tr>), а в строках — ячейки (<td>) или заголовки (<th>).

Структура таблицы

<table border="1">
 <caption>Заголовок таблицы</caption>
 <thead>
  <tr>
   <th>Столбец 1</th>
   <th>Столбец 2</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <td>Данные 1</td>
   <td>Данные 2</td>
  </tr>
 </tbody>
 <tfoot>
  <tr>
   <td colspan="2">Итог</td>
  </tr>
 </tfoot>
</table>

9. Введение в CSS

CSS (Cascading Style Sheets — каскадные таблицы стилей) — язык, описывающий внешний вид HTML-документа. Если HTML — это скелет страницы, то CSS — это её внешность.

Три способа подключения CSS

Способ Код Когда использовать
1. Встроенный (inline) <p style="color:red">...</p> Для одного элемента
2. Внутренний (internal) <style> p { color: red; } </style> Для одной страницы
3. Внешний (external) <link rel="stylesheet" href="style.css"> Для всего сайта ✓ рекомендуется

Синтаксис CSS-правила

селектор {
  свойство: значение;
  свойство2: значение2;
}

Примеры:
h1 { color: blue; font-size: 32px; }
p { font-family: Arial; line-height: 1.6; }
.my-class { background-color: #f0f0f0; }
#my-id { border: 1px solid red; }

📝 Задания для самостоятельной работы
  1. Базовая страница: Создать веб-страницу с заголовком, несколькими абзацами текста разного форматирования. Использовать только теги для работы с документом и текстом.
  2. Навигация: Реализовать веб-страницу с пунктами меню, боковой панелью и основной частью с текстом. Добавить ссылки, ведущие к нужным разделам документа.
  3. Медиа-страница: Создать страницу с несколькими изображениями разных размеров, встроить аудио- и видео-плеер с элементами управления.
  4. Таблица данных: Создать таблицу расписания занятий или прайс-лист с заголовком, шапкой (thead), телом (tbody) и итогами (tfoot). Использовать объединение ячеек colspan/rowspan.
  5. Форма регистрации: Создать форму с полями: имя, email, пароль, дата рождения, пол (radio), интересы (checkbox), город (select), комментарий (textarea), кнопки отправки и сброса.
  6. Полный сайт: Объединить всё изученное — создать сайт с шапкой, навигацией, боковой панелью, основным контентом (статьи, списки, таблицы, изображения, форма обратной связи) и подвалом.
